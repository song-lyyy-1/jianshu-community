# 简书问答社区 — 本地联调指南（Windows）

> 前置条件：JDK 17+、Maven、IDEA、HeidiSQL（你已有）
> 需额外安装：MySQL 8.x、Node.js 18+、pnpm

---

## 第一步：安装 MySQL（如果没有）

1. 下载 MySQL Installer：https://dev.mysql.com/downloads/installer/
2. 选择 **Custom** 安装，勾选：
   - MySQL Server 8.0
   - MySQL Workbench（可选，你有HeidiSQL就不需要）
3. 安装过程中设置 root 密码为 **root**（和项目配置一致）
4. 端口保持默认 **3306**
5. 安装完成后，打开 HeidiSQL 测试能否连接

> ⚠️ 如果你的MySQL密码不是root，需要修改 `jianshu-backend/src/main/resources/application.yml` 中的密码

---

## 第二步：初始化数据库

用 HeidiSQL 或命令行执行：

### 方式1：HeidiSQL（推荐，图形化更直观）
1. 打开 HeidiSQL，连接到你的MySQL
2. 点击 **文件 → 运行SQL文件**，选择项目中的：
   ```
   jianshu-community/jianshu-backend/src/main/resources/sql/init.sql
   ```
3. 点击执行，完成后左侧会出现 `jianshu` 数据库和5张表

### 方式2：命令行
```cmd
mysql -uroot -proot -e "CREATE DATABASE IF NOT EXISTS jianshu DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;"
mysql -uroot -proot jianshu < jianshu-backend\src\main\resources\sql\init.sql
```

验证：HeidiSQL中应该能看到：
- `jianshu` 数据库
- 5张表：user(3条)、article(5条)、article_like、article_favorite、comment
- 用户表里有 张三/李四/王五，密码都是 `test123`

## 第三步：安装 Node.js 和 pnpm（跑前端用）

### 1. 安装 Node.js
1. 下载：https://nodejs.org/ （选 **LTS版本**，当前是 v22.x）
2. 安装时全部 **Next** 即可，会自动配好环境变量
3. 安装完成后打开 **cmd** 验证：
   ```cmd
   node --version
   npm --version
   ```

### 2. 安装 pnpm
```cmd
npm install -g pnpm
pnpm --version
```

---

## 第四步：启动后端（IDEA）

### 1. 用IDEA打开后端项目
- File → Open → 选择 `jianshu-community/jianshu-backend` 文件夹
- 等待IDEA自动识别Maven项目并下载依赖（右下角进度条，可能需要1-3分钟）

### 2. 检查配置
打开 `src/main/resources/application.yml`，确认数据库连接信息：

```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/jianshu?useUnicode=true&characterEncoding=utf-8&useSSL=false&serverTimezone=Asia/Shanghai&allowPublicKeyRetrieval=true
    username: root
    password: root    # ← 如果你的MySQL密码不是root，改这里
```

### 3. 启动
- 找到 `src/main/java/com/jianshu/JianshuApplication.java`
- 右键 → **Run 'JianshuApplication'**
- 控制台出现 `Started JianshuApplication in x.xxx seconds` 就是成功了
- 后端运行在 **http://localhost:8080**

### 4. 验证后端
浏览器打开 http://localhost:8080/api/article/list?page=1&size=3
如果看到JSON数据返回，说明后端OK ✅

---

## 第五步：启动前端（VSCode或cmd）

### 1. 安装依赖
打开cmd，进入前端目录：
```cmd
cd jianshu-community\jianshu-frontend
pnpm install
```
等待安装完成（约1分钟）

### 2. 启动开发服务器
```cmd
npx vite --host 0.0.0.0 --port 5173
```
看到 `Local: http://localhost:5173/` 就是成功了

### 3. 验证前端
浏览器打开 **http://localhost:5173**
- 应该能看到简书问答首页，显示5篇博文
- 搜索图标、底部TabBar都能点击
- 点击博文卡片能进入详情页

> 前端的 `/api` 请求会自动代理到 `localhost:8080`（在 vite.config.js 中配置的）

## 常见问题排查

### ❌ 后端启动报数据库连接错误
- 检查MySQL是否启动了（Windows服务中找MySQL80）
- 检查 `application.yml` 中的密码是否和你的MySQL密码一致
- 检查是否执行了 `init.sql` 初始化数据库

### ❌ 后端启动报端口占用
```cmd
netstat -ano | findstr 8080
taskkill /PID <进程号> /F
```

### ❌ 前端 pnpm install 报错
- 确认Node.js版本 >= 18：`node --version`
- 清除缓存重试：`pnpm store prune && pnpm install`

### ❌ 前端页面空白或API报错
- 确认后端是否在 8080 端口运行
- 浏览器F12 → Network 标签，看请求是否到达后端
- 确认 `init.sql` 中的测试数据是否导入成功

### ❌ 登录报"密码错误"
- 确认你用的是 `test123` 作为密码
- 如果你自己改过数据库数据，密码是BCrypt加密的，不能手动改明文

### ❌ MySQL密码不是root怎么改？
打开 `jianshu-backend/src/main/resources/application.yml`，修改：
```yaml
spring:
  datasource:
    password: 你的MySQL密码
```

## 启动顺序总结

```
1. 启动 MySQL 服务
2. IDEA 启动后端（JianshuApplication.java）→ localhost:8080
3. cmd 启动前端（cd jianshu-frontend && npx vite）→ localhost:5173
4. 浏览器打开 http://localhost:5173
```

三个服务都启动后，前后端自动联通，直接体验即可 🎉
